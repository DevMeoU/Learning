🔑 Thông Tin Twilio

ℹ️ Thông tin cần thiết cho Twilio

  • Account SID: Tìm thấy trong bảng điều khiển Twilio của bạn
  • Auth Token: Mã bảo mật để xác thực tài khoản
  • From Number: Số điện thoại Twilio của bạn
  • Twilio URL: URL API của Twilio (mặc định: https://api.twilio.com)

🔗 Thông Tin AWS

ℹ️ Thông tin cần thiết cho AWS

  • AWS Endpoint: URL API Gateway hoặc dịch vụ AWS của bạn
  • URL này được sử dụng để gửi dữ liệu từ ESP32 lên AWS
  • Ví dụ: https://api-id.execute-api.region.amazonaws.com/stage/resource
